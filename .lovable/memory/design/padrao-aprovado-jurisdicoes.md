PADRÃO DE EXCELÊNCIA APROVADO (referência para todas as páginas pesadas):

1. HERO GIGANTE: imagem full-bleed ocupando 88vh+, NUNCA imagens pequenas ou médias na abertura. Imagens internas também devem ser grandes (560px+).

2. PALETA APROVADA (Jurisdições):
   - Sand: #f4ede4 (fundo claro, texto sobre escuro)
   - Deep Teal: #0e3b3a (fundo escuro)
   - Copper/Amber: #c4632a / #e8a36b / #ffb37a (acentos)
   Cada página pesada deve ter SUA própria paleta temática (3 cores), nunca tudo dark uniforme.

3. TIPOGRAFIA APROVADA:
   - Títulos impacto: Inter Tight 900, clamp(2.75rem, 8.5vw, 7.5rem), uppercase ou natural
   - Itálico editorial: Playfair Display italic com textShadow glow para contraste
   - PROIBIDO Bebas Neue em páginas editoriais pesadas

4. CONTRASTE OBRIGATÓRIO: textos de destaque (italic Playfair) precisam de textShadow glow para não sumirem no fundo. Nunca usar cor que se aproxime do fundo sem glow.

5. ALTERNÂNCIA: blocos claros (sand) e escuros (teal) alternados — NUNCA tudo dark, especialmente para público de idade avançada que lê no celular.

6. FONTES SEMPRE GRANDES: público sênior, mínimo text-lg/text-xl no body.

7. SEMPRE PREFERIR MAIS IMAGENS DO QUE MENOS — usuário aprovou e pediu ainda mais.
